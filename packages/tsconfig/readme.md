## @hellpack/tsconfig

_Ultracommon Tsconfig for Babel Compiled Typescript Projects_

#### Usage

- Install @hellpack/tsconfig `yarn add @hellpack/tsconfig -D`.
- Create a tsconfig file in your project root and add this

```javascript
{
  "extends": "@hellpack/tsconfig"
  // Your Settings here
}
```

- For example,

```javascript
{
  "extends": "@hellpack/tsconfig",
  "include": ["src"]
}
```
